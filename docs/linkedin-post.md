# LinkedIn Post

## Main Post

I just finished building and publishing CareerCommand, a desktop job application tracker designed for junior software engineers.

It helps track roles, companies, application stages, follow-ups, resume versions, skill matches, notes, and pipeline metrics in one focused dashboard.

This project was built to demonstrate the kind of work I want to do as a software engineer: turning a real user problem into a working, tested, and deployed application.

Tech stack:
- React
- TypeScript
- Vite
- Vitest
- Playwright
- GitHub Actions
- GitHub Pages

What I focused on:
- Building a clean desktop dashboard experience
- Modeling application data with TypeScript
- Adding search, filtering, add/edit workflows, and localStorage persistence
- Testing business logic with Vitest
- Testing the user flow with Playwright
- Shipping CI and a live deployment

Live demo:
